## Descripción

Proyecto para generar Screenshot, en el Videojuego de Grand Theft Auto: San Andreas hay servidores online donde la gente suele tomar fotos y generar imagenes con dialogos, recreando asi una historia ( Roleplay ) mi idea fue 
optimizar el tiempo de los jugadores, el cual les toma editando estas imagenes, que lo normal es en Photoshop. 

Optimizar para solo con seleccionar una imagen del texto, una imagen del fondo y pulsar en generar, ya tengamos la imagen editada, segun la estetica y formato de la comunidad de GTA San Andreas, de esta manera de paso hasta ahorramos tiempo y espacio instalando 
un programa como lo es photoshop. ( Aunque claro, si se desea tener mas opciones de edicion, claramente lo recomendado seria Instalar Photoshop ) 

---

## Tecnologías

- NextJS
- TypeScript
- React
- TailwindCSS

---

## Instalación

```bash
npm install

```
## Ejecutar
```bash
npm run dev
```
